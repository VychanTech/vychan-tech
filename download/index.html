Download VAI Studio

SHA-256 checksum - optional, recommended for verifying the download.

  1. Download `VychanAI-Studio.zip`.
  2. Optional: download the `.sha256.txt` checksum from the same release and compare it before installing.
  3. Right-click the zip and choose Extract All.
  4. Open the extracted folder.
  5. Double-click `Install VAI Studio.cmd`.
  6. Open VAI Studio from the Windows Start Menu folder.

VychanTech. VAI Studio does not store Twitch passwords, payment provider secrets, or Streamer.bot account credentials.
